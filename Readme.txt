PHP E-Commerce Website

This is a simple e-commerce web application built using PHP, MySQL, HTML,CSS.

-> Features

- User registration & login
- Admin login with product management
- Add to cart functionality
- Checkout with order summary
- Responsive UI using custom CSS
- Image upload for product photos


-> Technologies Used

- PHP
- MySQL (via XAMPP)
- HTML5, CSS3
- PDO for database interaction

-> Folder Structure
ecommerce/
├── admin/ Admin dashboard ,add_product,manage_product,logout,login
├── includes/ DB connection & utility files
├── pages/ Cart, login, checkout pages
├── uploads/ Product images
├── CSS/ Custom styles
├── index.php /Homepage
└── README.md
